# project-app
## Goals
<ul>
    <li>To offer financial services to end users like deposit, loan, and withdraw</li>
    <li>To offer financial information to new users such as pension schemes, stock shares and account details</li>
</ul>

## Target Audience
<ol>
    <li> Administrator</li>
    <li> Client</li>
    <li> Maintenance</li>
</ol>
